Flask blog

## Стек

- **Python 3.10+**
- **Flask 3.0** — веб-фреймворк
- **Flask-SQLAlchemy** — ORM для работы с БД
- **Flask-Login** — управление сессиями
- **Flask-WTF / WTForms** — формы с валидацией
- **Werkzeug** — хеширование паролей
- **Jinja2** — шаблонизатор
- **Bootstrap 5** — CSS-фреймворк для оформления
- **SQLite** — база данных


## Запуск

### Создайте виртуальное окружение и установите зависимости

**Linux / macOS:**

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt


**Windows:**

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

### 4. Запустите приложение


python run.py
