О проекте

Flask Blog это учебный блог, написанный в качестве итоговой работы по курсу python on web.

Стек технологий

  • Flask — веб-фреймворк, фабрика приложения, blueprint'ы
  • SQLite + SQLAlchemy — база данных
  • Flask-Login — управление сессиями пользователей
  • Flask-WTF / WTForms — формы с валидацией
  • Werkzeug — хеширование паролей
  • Jinja2 — шаблонизатор с наследованием шаблонов
  • Bootstrap 5 — CSS-фреймворк для оформления
